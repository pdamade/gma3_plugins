# GMA3 Plugins

This is a collection of plugins for grandMA3. All code is provided as is without any warranty, use at your own risk. Feedback is always appreciated. 

## CueList2Pdf
A simple plugin to export a given sequence in PDF format.

## ProjectorControl
A very specific plugin that sets up macros and a playback layout to send OSC commands to a host running Chataigne, which in turn controls a Digital Projection HighLite Laser 3D II video projector via RS232.
